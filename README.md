## Installation

```bash
# use to install the NPM packages from package-lock.json
$ npm ci
# use to install the NPM packages from package.json
$ npm install
```

## Configuration

```bash
# App configuration.
$ create .env file in main folder and addd all the required variable mentioned in '.dist.env' file
```

## Build

```bash
# Create a build.
$ npm run build
```

## Running the app

```bash
# Run Project.
$ npm run dev
```

## Running the Test

```bash
# Run test
$ npm run test
```
